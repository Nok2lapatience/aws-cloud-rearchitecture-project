# Deployment Steps

1. Created VPC networking components.
2. Provisioned Amazon RDS MySQL database.
3. Configured Amazon ElastiCache (Memcached).
4. Deployed RabbitMQ using Amazon MQ.
5. Stored deployment artifacts in Amazon S3.
6. Created Elastic Beanstalk application.
7. Configured environment variables.
8. Attached Application Load Balancer.
9. Configured HTTPS using ACM certificates.
10. Created CloudFront distribution.
11. Configured Route 53 DNS records.
12. Validated end-to-end application connectivity.
13. Performed testing and troubleshooting.
